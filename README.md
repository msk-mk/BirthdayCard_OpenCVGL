# BirthdayCard

色やデザインを選択し、写真・動画・音楽を組み合わせて、オリジナルの誕生日カードを作成・閲覧できるアプリケーションです。

アルバムのような自由な編集・閲覧体験、OpenGLによる3次元表現、複数のメディアを組み合わせたデジタルならではの演出を活かし、紙の誕生日カードでは表現できないインタラクティブなカード体験を実現します。

カードを作るためのプログラムと、完成したカードを閲覧するためのプログラムで構成しています。

## 特徴

- オリジナルのカード作成
  - 背景色やメインカラー、デザインを選びながら、贈る相手に合わせてカードをカスタマイズできます。
  - 宛名、差出人、写真、動画、音楽を指定し、思い出やメッセージを1つのカードにこめることができます。

- アルバムのようなページ閲覧
  - 完成したカードは複数ページで構成され、キー操作で前後へ移動できます。
  - 封筒、メッセージ、写真、動画、オブジェクトなどがページごとに切り替わり、ページをめくるごとに新しい表現が現れる体験となっています。

- 3D空間を活かしたカード表現
  - OpenGLによってオブジェクトを立体的に描画し、平面的なカードではなく空間として表現しています。
  - マウスドラッグによる視点回転やズームにより、様々な角度からカードを楽しめます。

- インタラクティブな閲覧体験
  - ボタン操作による写真表示や、オブジェクトの操作など、見るだけではなく触れながら楽しめます。
  - 隠されたメッセージを探索する仕掛けなど、ユーザの主体的な操作によって発見が生まれる仕掛けを設けています。

- デジタルメディアを組み合わせた表現
  - 写真や動画の表示、アニメーションによる画面演出の仕掛けがあります。
  - 音楽を組み合わせることで、視覚だけでなく聴覚も含めた誕生日カードを実現しています。

## ファイル構成

| ファイル | 役割 |
| --- | --- |
| `make.cpp` | カード作成用プログラム。画面の指示に従って入力内容を確認し、ユーザーが `template.txt` にカード情報を書き込むための補助ツールです。|
| `view.cpp` | カード閲覧用プログラム。`template.txt` を読み込み、指定された要素を使って完成カードを表示します。 |
| `template.txt` | カード設定ファイル。背景色、封筒色、名前、素材パスなどを管理します。 |

## `template.txt` の形式

`view.cpp` は `template.txt` を読み込み、以下の行をカード情報として扱います。

| 行 | 内容 | 例 |
| --- | --- | --- |
| 1 | 背景色番号 | `1` |
| 2 | 基本色番号 | `2` |
| 3 | 宛名 | `Mr.Bear` |
| 4 | 差出人 | `Rabbit` |
| 5 | 音楽ファイル名 | `song.wav` |
| 6-10 | 写真ファイル名 | `photo.jpg` |
| 11 | メインデザイン番号 | `1` |
| 12 | 動画ファイル名 | `movie.mp4` |

## 技術スタック

- C++
- OpenGL
- GLUT
- OpenCV
- SFML Audio

## 使い方

1. `make.cpp` を起動する

2. 画面の案内を確認しながら、カードに使用する情報を `template.txt` に入力する

3. `view.cpp` を起動し、完成したカードを閲覧する

4. キー操作やマウス操作によって、ページ切り替えや3D空間での閲覧を楽しむ
