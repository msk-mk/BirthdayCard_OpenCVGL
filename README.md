# BirthdayCard

色やデザインを選択し、写真・動画・音楽を組み合わせて、オリジナルの誕生日カードを作成・閲覧できるアプリケーションです。

複数のメディアを組み合わせたデジタルならではの表現や、OpenGLによる3次元表現を取り入れることで、紙の誕生日カードでは表現できないインタラクティブなカード体験を実現します。

本アプリケーションは、カードを作成するプログラムと、完成したカードを閲覧するプログラムの2つで構成されています。

## 特徴

- オリジナルのカード作成
  - 背景色やメインカラー、デザインを選びながら、贈る相手に合わせてカードをカスタマイズできます。
  - 宛名・差出人・写真・動画・音楽を指定し、思い出やメッセージを1つのカードにこめることができます。

- アルバムのようなページ閲覧
  - 完成したカードは複数ページで構成され、キー操作でページを切り替えることができます。
  - ページごとに封筒・メッセージ・写真・動画・3Dオブジェクトなどの要素が表示され、ページをめくるごとに新しい表現が現れる体験となっています。

- 3D空間を活かしたカード表現
  - OpenGLによる3次元表現を取り入れ、平面のカードを超えた動きや奥行きを感じられる表現を実現しています。
  - マウスドラッグによる視点回転やズームにより、様々な角度からカードを楽しめます。

- インタラクティブな閲覧体験
  - ボタン操作による写真表示やオブジェクト操作など、見るだけではなく触れながらカードを楽しむことができます。
  - 隠されたメッセージの探索など、ユーザー自身の操作によって新しい発見が生まれる仕掛けを取り入れています。

- デジタルメディアを組み合わせた表現
  - 写真や動画の表示、アニメーションによる演出を組み合わせ、紙のカードでは表現できない動的な表現を実現しています。
  - 音楽を組み合わせることで、視覚だけでなく聴覚も含めた誕生日カードを作成できます。

## ファイル構成

| ファイル | 役割 |
| --- | --- |
| `make.cpp` | カード作成用プログラム。画面の指示に従って入力内容を確認し、ユーザーが `template.txt` にカード情報を書き込むための補助ツールです。|
| `view.cpp` | カード閲覧用プログラム。`template.txt` を読み込み、指定された要素を使って完成カードを表示します。 |
| `template.txt` | カード設定ファイル。背景色、封筒色、名前、素材パスなどを管理します。 |

### `template.txt` の形式

`view.cpp` は `template.txt` を読み込み、以下の行をカード情報として扱います。

| 行 | 内容 | 例 |
| --- | --- | --- |
| 1 | 背景色番号 | `1` |
| 2 | 基本色番号 | `2` |
| 3 | 宛名 | `Mr.Bear` |
| 4 | 差出人 | `Rabbit` |
| 5 | 音楽ファイル名 | `song.wav` |
| 6-10 | 写真ファイル名 | `photo.jpg` |
| 11 | メインデザイン番号 | `1` |
| 12 | 動画ファイル名 | `movie.mp4` |

## 技術スタック

- C++
- OpenGL
- GLUT
- OpenCV
- SFML Audio

## 体験の流れ

1. `make.cpp` を起動します。

2. 画面の案内に従って、カードに使用する情報を `template.txt` に入力します。

3. `view.cpp` を起動すると、作成したカードを閲覧できます。

4. キー操作やマウス操作によって、ページ切り替えやオブジェクト操作などのインタラクティブな閲覧を楽しめます。
